# Checkout System

## Table of Contents

- [Install](#install)
- [How to Use](#how-to-use)

This system is built on [Nest](https://nestjs.com/) framework.

## Installation

```bash
$ npm install
```

Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

Running Tests

```bash
# unit tests
$ npm run test
```

## How to use

Visit http://localhost:3000/api in your browser.
